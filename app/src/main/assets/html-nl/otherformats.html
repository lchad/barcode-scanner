Barcode Scanner+ kan scannen vele andere formaten, die minder vaak voor, maar nog steeds gebruikt in specifieke contexten, zoals vervoerbewijzen of industriële deel etikettering. Scannen voor elk van deze formaten kunnen worden ingeschakeld of uitgeschakeld in instellingen, ze zijn met een handicap standaard.

PDF417

PDF417 wordt meestal gebruikt door pakketdienst bedrijven voor de scheepvaart informatie te coderen. Merk op dat het moeilijk is om PDF417 lezen van een mobiel. U kunt kiezen voor hoge resolutie preview in instellingen, ter verbetering van het decoderen van deze indeling mogelijk te maken.

Aztec

Aztec barcodes zijn vooral te vinden op tickets voor treinen en vliegtuigen.

Industriële 1D-indelingen

Ten slotte, Barcode Scanner+ kunnen lezen verschillende 1D-formaten die hoofdzakelijk worden gebruikt in de commerciële of industriële toepassingen. Boven is een voorbeeld van Code 39, ze kan moeilijk zijn om visueel te onderscheiden. De applicatie kan lezen:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

In het bijzonder kan deze weer te geven wat informatie over VIN (Vehicle Identification Number) barcodes.
