Barcode Scanner+ ce n'est pas seulement recevoir informations à partir des codes à barres sur le marché. Il peut aussi Envoyer l'information, en créant des codes-barres que les autres peuvent scanner hors de l'écran de votre appareil - peut-être une autre personne avec un appareil mobile comme la vôtre. Les codes à barres peuvent être un moyen pratique pour transférer de petites quantités d'informations entre les périphériques.

Partage de l'information

Il suffit d'appuyer sur Menu, puis choisissez «Partager». Sur l'écran suivant, vous serez capable de choisir quel type d'information que vous souhaitez partager.

Applications

Pour n'importe quelle application sur votre appareil, vous pouvez générer un code à barres qui encode un lien dans la page de cette application dans l'Android Market. Lorsque scannés par un autre appareil Android, il permettra à l'utilisateur de télécharger rapidement de la demande.

Favoris

De même, vous pouvez choisir un signet à partir navigateurs de l'appareil, et il part comme un lien hypertexte codées dans un code-barres. Un autre appareil peut scanner les codes barres et ensuite accéder au lien hypertexte.

Contactez

Toute information de contact à partir de votre appareil peut être encodée dans un code-barres. Lorsque numérisées à partir d'un autre appareil, il doit permettre à l'utilisateur d'enregistrer ce contact dans la base de son appareil propre contacts.

Vous pouvez choisir le format vCard pour des informations de contact au lieu du format par défaut MECARD en appuyant sur Menu en premier.

Presse-papiers

Vous pouvez encoder ce qui est sur le presse-papiers de l'appareil pour le moment. Ce sera désactivée si le presse-papiers est vide.

Texte

Enfin, vous pouvez simplement taper du texte dans le champ de texte au bas de l'écran. Appuyez sur la touche Entrée pour encoder.

Enregistrement des codes à barres générés

Vous pouvez enregistrer les codes-barres que vous générez avec cette application. Lorsque vous visualisez le code-barres généré sur l'écran, appuyez sur Menu, puis sélectionnez "Partager". Comme avec l'exportation de l'histoire de balayage , Vous pouvez alors choisir un programme avec lequel vous voulez stocker ou envoyez l'image, comme une application de messagerie électronique.

L'image est enregistrée au format PNG. Notez que la copie du fichier d'image est aussi écrit sur la carte SD de votre appareil dans un répertoire appelé " BarcodeScanner/Barcodes ».
