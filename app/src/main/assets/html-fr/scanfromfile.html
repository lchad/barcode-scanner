Il est possible de décoder un code-barres qui est stocké dans un fichier image sur votre appareil, plutôt que de numériser avec la caméra de l'appareil. Pour ce faire, appuyez sur Menu et choisissez «Image Decode".

Vous serez invité à choisir une application sur votre appareil qui traite les images, comme l'application Galerie ou un gestionnaire de fichiers. Choisir un fichier image contenant un code-barres en utilisant cette application. Vous pourrez alors revenir à la Barcode Scanner+ .

En cas de succès, vous verrez un écran de résultat comme si vous aviez un code à barres scannés à l'aide de la caméra. Si non, vous verrez un court message d'erreur.
