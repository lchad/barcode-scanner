Barcode Scanner+ stocke une histoire de codes-barres que vous avez scanné. Cela inclut le contenu, le temps, le format, et d'autres renseignements connexes. Vous pouvez accéder à l'historique en appuyant sur le bouton Menu, puis en sélectionnant l'Histoire.

Vous verrez une liste des analyses précédentes. Appuyez sur l'une de rappeler les détails de l'analyse à l'écran, et agir sur elle comme si elle avait été scanné à nouveau. Longue appuyez sur pour effacer un article. Vous pouvez effacer l'historique en appuyant sur Menu, puis en sélectionnant "Effacer l'historique".

L'histoire Exportation

Vous pouvez exporter les données dans votre historique de numérisation, de sorte qu'il peut être lu et traité par une application externe. Les données sont exportées au format de valeur séparé par des virgules (CSV). Les colonnes de sortie, dans l'ordre, sont:

  • Contenu brut
  • Afficher le contenu formaté
  • Format (par exemple, QR_CODE)
  • Horodatage, en millisecondes depuis l'époque (par exemple, 1315175453569)
  • Timestamp en forme (par exemple, le 4 septembre 2011 23:30:53)
  • Autres informations

Appuyez sur Menu et choisissez "Envoyez l'histoire". Vous serez capable de choisir à partir d'applications sur votre appareil qui sont en mesure d'envoyer les données quelque part, comme une application de messagerie électronique avec lequel vous pouvez envoyer les données CSV à votre adresse e-mail. Notez qu'après l'envoi des données, il sera également enregistré sur votre carte SD dans un répertoire appelé " BarcodeScanner/History ».

Copie dans le presse papier

Par défaut, le contenu de chaque balayage sont copiés dans le presse-papiers aussi. Pour désactiver cela, décochez la case "Copy to clipboard" dans les Paramètres.

Vrac mode de balayage

Certains utilisateurs ont besoin de scanner des codes barres nombreuses, afin de stocker et plus tard l'exportation de l'histoire de balayage. Pour rendre cela plus facile, activez l'option "mode de numérisation en vrac" sous Paramètres. Après chaque scan avec succès, l'application va se préparer à lire un code à barre, au lieu d'afficher l'écran de résultat.

Numérisations multiples d'un code-barres

Par défaut, si un code-barres est scanné plusieurs fois, l'histoire n'enregistre que l'analyse la plus récente de ce code à barres. Ceci peut être changé en permettant «Rappelez-vous les doublons" dans les Paramètres. Cela entraînera l'histoire d'enregistrer toutes les analyses.
