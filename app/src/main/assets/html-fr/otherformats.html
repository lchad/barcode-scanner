Barcode Scanner+ permet de numériser de nombreux autres formats, qui sont moins fréquents mais toujours utilisés dans des contextes spécifiques, comme des billets de voyage ou l'étiquetage partie industrielle. Numérisation pour chacun de ces formats peut être activée ou désactivée dans les paramètres, ils sont disabled par défaut.

PDF417

PDF417 est habituellement utilisé par les entreprises de livraison de colis pour coder les informations de livraison. Notez qu\'il est difficile à lire PDF417 partir d\'un périphérique mobile. Vous pouvez choisir d\'activer aperçu haute résolution dans les paramètres, ce qui permettra d\'améliorer le décodage de ce format.

Aztec

Codes à barres aztèques se trouvent principalement sur les billets pour les trains et les avions.

Industrielle formats 1D

Enfin, Barcode Scanner+ peut lire plusieurs formats 1D différents qui sont utilisés principalement dans des applications commerciales ou industrielles. Ci-dessus est un exemple de code 39, ils peuvent être difficiles à distinguer visuellement. L'application peut lire:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

En particulier, il peut afficher des informations sur VIN (Vehicle Identification Number) des codes barres.
