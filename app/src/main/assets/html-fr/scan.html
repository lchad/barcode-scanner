Pour numériser, il suffit de mettre le code à barres en vue dans le centre de l'écran de votre appareil. Barcode Scanner+ va essayer de détecter automatiquement et de décoder les codes-barres, vous n'avez pas besoin d'appuyer sur un bouton.

Vous verrez des points jaunes apparaissent à l'écran. Ces données indiquent que l'application pense qu'elle a trouvé une partie d'un code à barres dans l'image sur l'écran. Quand il décode avec succès un code à barres complet, vous entendrez un bip et / ou des vibrations, et l'écran affichera les résultats. (Vous pouvez activer ou désactiver le bip sonore et les vibrations sous Paramètres.)

L'écran des résultats affichera informations supplémentaires pour les produits , Et présentera des boutons d'action qui peut être utilisée pour agir sur la le contenu du code à barres .

Ne tenez pas le code à barres trop près de la caméra, il ne peut se concentrer que près. Le code-barres ne doivent pas remplir l'écran. Vous verrez un exemple d'image transparente de codes à barres dans le centre de l'écran. Aligner le code à barres à ce guide pour s'assurer qu'elle est à la distance proposée.

Lors de la numérisation carrés, des formats 2D comme QR Code, Data Matrix et Aztec, aligner à l'exemple de carré 2D que vous voyez en bleu clair. Lors de la numérisation des formats larges 1D comme le CUP et les codes EAN des produits, d'harmoniser à l'exemple de code à barres 1D large en rose. Ces deux guides seront s'affichent alternativement.

Caméras avant

Certains appareils, comme les ordinateurs tablettes, ont une caméra frontale supplémentaire. Sélectionnez "Préférez caméra frontale" sous Paramètres de scanner avec elle. Habituellement, il est seulement possible de numériser des formats 2D avec cet appareil.

Blanc sur noir codes barres

Les codes barres sont généralement imprimés en noir sur un fond blanc. Certains, cependant, sont imprimés en blanc sur fond noir. Pour numériser celles-ci, activez l'option "Inverser la numérisation» sous Paramètres. Vous devriez voir une image vidéo inversée. Notez que cette fonction n'a aucun effet sur certains appareils.

Numérisation en basse lumière

Il est difficile de scanner des codes barres dans l'éclairage est faible ou irrégulier. Certains appareils ont une lumière, qui est utilisé comme un flash pour prendre des photos. Cela peut être allumé en permanence pour éclairer le code à barres scanné. Sélectionnez "Utiliser la lumière de front» sous Paramètres pour permettre à la lumière, ou, appuyez sur la touche Augmenter volume sur l'appareil s'il en a un. Pas tous les appareils ont une lumière qui peut être activé. Notez que la lumière peut provoquer des reflets qui rend également la numérisation difficile.

Utiliser l'auto-focus

Par défaut, l'application utilise la caméra de votre appareil autofocus fonction, le cas échéant, afin de scanner des codes barres mieux. Cela devrait être laissé activé, en général. Cependant, il est possible de désactiver l'auto-focus en décochant "auto focus utilisation» dans les Paramètres.
