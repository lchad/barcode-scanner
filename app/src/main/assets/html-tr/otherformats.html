Barkod Tarayıcısı+ seyahat biletleri veya endüstriyel parçası etiketleme gibi belirli bağlamlarda daha az yaygın fakat halen kullanılmakta olan pek çok diğer formatlar, tarama yapabilirsiniz. Bu biçimlerin her Ayarlar etkin veya devre dışı bırakılabilir. Tarama için; özürlü varsayılan olarak.

PDF417

PDF417 genellikle denizcilik bilgileri kodlamak için paket teslimat şirketleri tarafından kullanılır. Bir mobil cihazdan PDF417 okunması zor olduğunu unutmayın. Bu biçimde çözme artıracak Ayarlar yüksek çözünürlüklü önizleme, etkinleştirmeyi seçebilirsiniz.

Aztec

Aztek barkod öncelikle trenler ve uçaklar için biletlerini bulunur.

Endüstriyel 1D formatları

Sonuç olarak, Barkod Tarayıcısı+ öncelikle ticari veya endüstriyel uygulamalarda kullanılan birçok farklı 1D formatları okuyabilirsiniz. Yukarıda bir örnek Code 39, görsel olarak ayırt etmek zor olabilir. Uygulama okuyabilirsiniz:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Özellikle, VIN (Araç Tanımlama Numarası) barkodlar ile ilgili bazı bilgileri görüntüleyebilirsiniz.
