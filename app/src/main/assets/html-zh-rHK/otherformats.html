條碼掃描器+ 可以掃描多種格式，這是不常見，但在諸如工業旅遊機票或部分標示具體情況仍在使用。掃描這些格式都可以在設置中啟用或禁用，它們是 禁用 默認情況下。

PDF417

通常用來包裹運送公司的PDF417編碼航運信息。請注意，這是很難移動設備閱讀PDF417碼。你可以選擇啟用在設置的高分辨率預覽，這將改善這種格式的解碼。

Aztec

阿茲台克條形碼主要發現火車和飛機的門票。

工業 1D格式

最後， 條碼掃描器+ 可以讀取幾種不同的1D，用於商業或工業應用的主要格式。上面是一個 39碼的例子，他們可能很難區分視覺。應用程序可以讀取：

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

特別是，它可以顯示一些有關 VIN（車輛識別號碼）條碼的信息。
