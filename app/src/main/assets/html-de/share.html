Barcode Scanner+ geht nicht nur um Empfang Informationen von Barcodes gibt. Es kann auch senden Informationen, durch die Schaffung von Barcodes, die anderen können Scans von Ihrem Gerät auf den Bildschirm - vielleicht eine andere Person mit einem mobilen Gerät wie dem Ihrigen. Barcodes können eine bequeme Möglichkeit, kleine Mengen von Informationen zwischen den Geräten übertragen werden.

Die Weitergabe von Informationen

Drücken Sie einfach Menü, dann wählen Sie "Senden". Auf dem folgenden Bildschirm können Sie entscheiden, welche Art von Informationen, die Sie freigeben möchten.

Anwendungen

Für jede Anwendung auf Ihrem Gerät, können Sie einen Barcode, der einen Link in die Anwendung auf der Seite im Android Market kodiert. Wenn von einem anderen Android-Gerät gescannt, wird es dem Benutzer erlauben, schnell das Anmeldeformular herunterladen.

Bookmarks

Ebenso können Sie ein Lesezeichen aus dem Gerät Browsern, und teilen Sie sie als Hyperlink in einem Barcode verschlüsselt. Ein weiteres Gerät scannen kann den Barcode und dann auf den Hyperlink.

Kontakt

Jeglicher Kontakt Informationen von Ihrem Gerät kann in einem Barcode verschlüsselt werden. Wenn von einem anderen Gerät gescannt, sollte es dem Benutzer erlauben, dass der Kontakt in seinem eigenen Gerät die Kontakte-Datenbank zu speichern.

Sie können die vCard Format für Kontaktinformationen anstelle des Standard MECARD Format wählen, indem Sie zunächst Menü drücken.

Zwischenablage

Sie können kodieren, was auf dem Gerät die Zwischenablage im Moment. Dies wird deaktiviert, wenn die Zwischenablage leer ist.

Text

Schließlich können Sie ganz einfach Text in das Textfeld am unteren Rand des Bildschirms. Drücken Sie die Eingabetaste, um zu kodieren.

Saving erzeugten Barcodes

Sie können die Barcodes erzeugen Sie mit dieser Anwendung zu speichern. Beim Betrachten der erzeugten Barcodes auf dem Bildschirm, drücken Sie Menü und wählen Sie dann "Senden". Wie bei Export der Scan-Geschichte Sie können wählen Sie dann ein Programm, mit dem zu speichern oder senden Sie das Bild, wie eine E-Mail-Anwendung.

Das Bild wird im PNG-Format gespeichert. Beachten Sie, dass Sie eine Kopie der Bilddatei auch auf Ihrem Gerät eine SD-Karte in einem Ordner namens geschrieben " BarcodeScanner/Barcodes ".
