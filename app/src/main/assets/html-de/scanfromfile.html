Es ist möglich, einen Barcode, der in einer Image-Datei auf Ihrem Gerät gespeichert ist, anstatt scannen mit dem Gerät die Kamera zu entschlüsseln. Dazu drücken Sie Menü und wählen Sie "Decode Image".

Sie werden aufgefordert, eine Anwendung auf Ihrem Gerät die Bilder, wie die Galerie-Anwendung oder eines Dateimanagers Griffen wählen. Wählen Sie eine Bilddatei mit einem Barcode mit dieser Anwendung. Sie werden dann zur Rückkehr Barcode Scanner+ .

Wenn dies gelingt, werden Sie sehen, ein Ergebnis Bildschirm gerade, als hätten Sie einen Barcode mit der Kamera gescannt. Wenn nicht, sehen Sie eine kurze Fehlermeldung.
