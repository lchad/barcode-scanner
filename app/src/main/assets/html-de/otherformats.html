Barcode Scanner+ können viele andere Formate, die weniger häufig, aber immer noch verwendet in bestimmten Kontexten wie Fahrkarten oder industriellen Teil Kennzeichnung sind zu scannen. Scanning für jedes dieser Formate kann aktiviert oder in-Einstellungen deaktiviert, sie sind Behinderte standardmäßig aktiviert.

PDF417

PDF417 ist in der Regel durch Paketdienst-Unternehmen verwendet werden, um Versandkosten zu kodieren. Beachten Sie, dass es schwierig ist, PDF417 von einem mobilen Gerät zu lesen. Sie können wählen, hochauflösende Vorschau in den Einstellungen, die Dekodierung von diesem Format zu verbessern ermöglicht.

Aztec

Aztec Barcodes werden in erster Linie auf Tickets für Züge und Flugzeuge gefunden.

Industrielle 1D-Formate

Schließlich Barcode Scanner+ lesen können verschiedene 1D-Formate, die hauptsächlich in gewerblichen oder industriellen Anwendungen eingesetzt. Oben ist ein Beispiel für Code 39, sie kann hart sein, um visuell zu unterscheiden. Die Anwendung kann man lesen:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Insbesondere kann es zeigt einige Informationen über VIN (Vehicle Identification Number) Barcodes.
