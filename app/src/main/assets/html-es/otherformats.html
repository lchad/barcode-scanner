Barcode Scanner+ puede explorar otros formatos, que son menos comunes pero que se utilizan aún en contextos específicos, como billetes de transporte o el etiquetado parte industrial. La exploración de cada uno de estos formatos se pueden activar o desactivar en la configuración, sino que son discapacitado por defecto.

PDF417

PDF417 es utilizado generalmente por las compañías de entrega de paquetes para codificar la información de envío. Nótese que es difícil de leer PDF417 de un dispositivo móvil. Usted puede optar por habilitar vista previa de alta resolución en la configuración, lo que mejorará la decodificación de este formato.

Aztec

Códigos de barras Aztec se encuentran principalmente en los billetes de los trenes y aviones.

Industrial formatos 1D

Por último, Barcode Scanner+ puede leer varios formatos 1D diferentes que se utilizan principalmente en aplicaciones comerciales o industriales. Lo anterior es un ejemplo de código 39, ya que pueden ser difíciles de distinguir visualmente. La aplicación puede leer:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

En particular, se puede mostrar alguna información acerca de VIN (Vehicle Identification Number) de códigos de barras.
