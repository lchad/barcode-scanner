Es posible descifrar un código de barras que se almacena en un archivo de imagen en el dispositivo, en lugar de escanear con la cámara del dispositivo. Para hacerlo, pulse Menú y seleccione "Imagen Decode".

Se le pedirá que elija una aplicación en el dispositivo que se encarga de las imágenes, tales como la aplicación de la Galería o un administrador de archivos. Elija un archivo de imagen que contiene un código de barras mediante la aplicación. A continuación, volver a Barcode Scanner+ .

Si tiene éxito, usted verá una pantalla de resultados como si se escanea un código de barras con la cámara. Si no, aparecerá un mensaje de error corto.
