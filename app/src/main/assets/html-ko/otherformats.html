바코드 스캐너+ 여행 항공권 또는 산업 부분 라벨과 같은 특정 상황에서 많이 사용하지만 여전히 사용되는 다른 많은 형식을 검색할 수 있습니다. 이러한 형식의 각각 설정에서 활성화 또는 비활성화할 수에 대한 검사, 그들은 장애인 기본적으로.

PDF417

PDF 417은 일반적으로 배송 정보를 인코딩하는 패키지 공급 업체에 의해 사용됩니다. 그것이 모바일 장치에서 PDF 417을 읽고 어려운 있습니다. 이 포맷의 디코딩을 향상 설정 고해상도 미리보기를 사용하도록 선택할 수 있습니다.

Aztec

아즈텍 바코드는 주로 기차와 비행기 티켓을 볼 수 있습니다.

산업 1D 형식

마지막으로, 바코드 스캐너+ 상업용 또는 산업용 애플 리케이션에 주로 사용되는 몇 가지 1D 형식을 읽을 수 있습니다. 위의 코드는 39의 예입니다, 그들은 시각적으로 구분하기 힘들 수 있습니다. 응용 프로그램이 읽을 수 있습니다 :

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

특히, 그것은 VIN (차량 식별 번호) 바코드에 대한 몇 가지 정보를 표시할 수 있습니다.
