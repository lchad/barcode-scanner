Barcode Scanner+ कई अन्य स्वरूपों, जो यात्रा टिकट या औद्योगिक हिस्सा लेबलिंग की तरह विशिष्ट संदर्भों में कम आम है लेकिन अभी भी प्रयोग किया है स्कैन कर सकते हैं. के लिए इन स्वरूपों में से प्रत्येक या सक्षम किया जा सकता है सेटिंग्स में अक्षम स्कैन, वे कर रहे हैं विकलांग डिफ़ॉल्ट रूप से.

PDF417

PDF417 आम तौर पर पैकेज वितरण कंपनियों द्वारा प्रयोग किया जाता है शिपिंग जानकारी सांकेतिक शब्दों में बदलना. ध्यान दें कि यह मुश्किल है के लिए एक मोबाइल डिवाइस से PDF417 पढ़ने के. आप उच्च संकल्प पूर्वावलोकन सेटिंग्स में, जो इस प्रारूप के decoding में सुधार होगा सक्षम करना चुन सकते हैं.

Aztec

एज़्टेक बारकोड को मुख्य रूप से ट्रेनों और हवाई जहाज के लिए टिकटों पर पाए जाते हैं.

औद्योगिक 1D स्वरूपों

अंत में, Barcode Scanner+ कई अलग अलग 1D प्रारूपों है कि मुख्य रूप से वाणिज्यिक या औद्योगिक अनुप्रयोगों में उपयोग किया जाता है पढ़ सकते हैं. ऊपर 39 कोड का एक उदाहरण है, वे नेत्रहीन भेद करना मुश्किल हो सकता है. आवेदन पढ़ने के कर सकते हैं:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

विशेष रूप में, यह VIN (वाहन पहचान संख्या) बारकोड के बारे में कुछ जानकारी प्रदर्शित कर सकते हैं.
