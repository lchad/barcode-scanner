Barcode Scanner+ kan skanne mange andre formater, som er mindre vanlig, men fortsatt brukes i bestemte sammenhenger som for billetter eller industrielle delen merking. Skanning for hver av disse formatene kan aktiveres eller deaktiveres i Innstillinger, de er deaktivert som standard.

PDF417

PDF417 er vanligvis brukt av pakken levering selskaper å kode fraktinformasjon. Merk at det er vanskelig å lese PDF417 fra en mobil enhet. Du kan velge å aktivere høy oppløsning forhåndsvisning i Innstillinger, noe som vil forbedre dekoding av dette formatet.

Aztec

Aztec strekkoder er først og fremst funnet på billetter til tog og fly.

Industrielle 1D formater

Endelig, Barcode Scanner+ kan lese flere forskjellige 1D formater som brukes primært i kommersielle eller industrielle applikasjoner. Ovenfor er et eksempel på kode 39, de kan være vanskelig å skille visuelt. Søknaden kan lese:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Spesielt kan det vise litt informasjon om VIN (Vehicle Identification Number) strekkoder.
