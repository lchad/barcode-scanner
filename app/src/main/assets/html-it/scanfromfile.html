E 'possibile decodificare un codice a barre che viene memorizzato in un file di immagine sul tuo dispositivo, invece di eseguire la scansione con la fotocamera del dispositivo. Per farlo, premete Menu e scegliere "Immagine Decode".

Ti verrà richiesto di scegliere un'applicazione sul dispositivo che gestisce le immagini, come ad esempio l'applicazione Galleria o un file manager. Scegli un file immagine che contiene un codice a barre utilizzando tale applicazione. Si torna quindi a Barcode Scanner+ .

Se avrà successo, verrà visualizzata una schermata dei risultati come se fosse un codice a barre scansionati con la fotocamera. In caso contrario, verrà visualizzato un breve messaggio di errore.
