Barcode Scanner+ käyttää laitteesi kameraa kaikenlaisten viivakoodin lukemiseen (skannaamiseen). Esimerkiksi: jos luet viivakoodin tuotepakkauksesta, voit etsiä hintatietoja ja tuotearvosteluita tuosta tuotteesta. Uudempia kaksiuloitteisia viivakoodeja - kuten QR Code - skannamalla saat luettua kontaktitietoja, kirjanmerkkejä, www-osoitteita ja monia muita tietoja.
