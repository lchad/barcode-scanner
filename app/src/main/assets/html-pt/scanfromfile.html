É possível decodificar um código de barras que é armazenado em um arquivo de imagem em seu dispositivo, em vez de varredura com a câmera do dispositivo. Para isso, pressione Menu e escolha "Image Decode".

Você será solicitado a escolher uma aplicação no dispositivo que manipula as imagens, tais como a aplicação Galeria ou um gerenciador de arquivos. Escolha um arquivo de imagem contendo um código de barras usando o aplicativo. Então você vai voltar Barcode Scanner+ .

Se for bem sucedido, você verá uma tela de resultado como se você tivesse verificado um código de barras usando a câmera. Se não, você verá uma mensagem de erro curto.
