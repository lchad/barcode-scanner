Para digitalizar, basta levar o código de barras em exibição no centro da tela do seu dispositivo. Barcode Scanner+ tentará detectar automaticamente e decodificar o código de barras, você não precisa pressionar um botão.

Você vai ver pontos amarelos aparecem na tela. Estes indicam que a aplicação acho que ele foi encontrado parte de um código de barras na imagem na tela. Quando decodifica um código de barras com sucesso completo, você ouvirá um sinal sonoro e / ou vibração, ea tela irá mostrar resultados. (Você pode ativar ou desativar o sinal sonoro e vibração em Configurações.)

A tela irá mostrar resultado informações adicionais para produtos , E irá apresentar os botões de ação que podem ser usados para dar seguimento às conteúdo do código de barras .

Não segure o código de barras muito perto da câmera, mas não pode focar tão perto. O código de barras não deve preencher a tela. Você verá uma imagem transparente de código de barras exemplo, no centro da tela. Alinhar o código de barras para este guia para garantir que está a uma distância sugerida.

Ao digitalizar quadrados, formatos 2D, como QR Code, Data Matrix e asteca, alinhar ao exemplo quadrado 2D você vê em azul claro. Ao digitalizar ampla formatos 1D como UPC e EAN códigos de produtos, alinhe com o exemplo maior de códigos de barras 1D em rosa. Estes dois guias serão exibidos alternadamente.

Câmeras frontal

Alguns dispositivos, como computadores tablet, tem uma câmera frontal adicional. Selecione "Preferir câmera frontal" em Configurações para fazer a varredura com ele. Normalmente, só é possível digitalizar formatos 2D com esta câmara.

Branco sobre preto códigos de barras

Códigos de barras são geralmente impressos em preto sobre um fundo branco. Alguns, no entanto, são impressos em branco sobre fundo preto. Para digitalizar esses, ative "Inverter scan" em Configurações. Você deverá ver uma imagem de vídeo invertido. Note que esta função não tem efeito sobre alguns dispositivos.

Digitalização em baixa luminosidade

É difícil ler códigos de barras em baixa iluminação ou irregular. Alguns dispositivos têm uma luz, que é usado como um flash para tirar fotos. Isto pode ser ligado continuamente para iluminar o código de barras que está sendo digitalizado. Selecione "Use a luz de frente" em Configurações para habilitar a luz, ou, pressione a tecla Aumentar volume no dispositivo se ele tiver um. Nem todos os dispositivos têm uma luz que pode ser ativado. Note-se que a luz pode causar reflexos que também facilita a digitalização difícil.

Usando o foco automático

Por padrão, o aplicativo irá utilizar o dispositivo de câmera recurso de auto-foco, onde disponíveis, a fim de ler códigos de barras melhor. Este deve ser deixado habilitado, em geral. No entanto, é possível desabilitar auto-foco, desmarcando "foco automático Use" em Configurações.
