Barcode Scanner+ handlar inte bara om emot information från streckkoder där ute. Det kan också skicka information, genom att skapa streckkoder som andra kan söka från din enhetens skärm - kanske en annan person med en mobil enhet som ditt. Streckkoder kan vara ett bekvämt sätt att överföra små mängder av information mellan enheter.

Dela information

Bara att trycka på Meny och välj sedan "Dela". På följande skärm, kommer du att kunna välja vilken typ av information du vill dela.

Applikationer

För alla program på enheten kan du generera en streckkod som kodar en länk till det programmet sida på Android Market. När skannas av en annan Android-enhet, kommer det göra det möjligt för användaren att snabbt ladda ner programmet.

Bokmärken

Likaså kan du välja ett bokmärke från enhetens browers och dela det som en hyperlänk kodad i en streckkod. En annan enhet kan skanna streckkoden och sedan komma på hyperlänken.

Kontakt

All kontakt information från din enhet kan kodas in i en streckkod. När scannade från en annan enhet, bör det möjligt för användaren att spara den kontakten i hans eller hennes egna enhetens kontakter databas.

Du kan välja vCard-formatet för kontaktinformation istället för standard MECARD format genom att först trycka på Menu.

Urklipp

Du kan koda vad som finns på enhetens Urklipp för tillfället. Detta kommer att inaktiveras om klippbordet är tomt.

Text

Slutligen kan du helt enkelt skriva texten i textfältet längst ner på skärmen. Tryck på Enter för att koda.

Spara genererade streckkoder

Du kan spara streckkoder du genererar med denna ansökan. När du tittar på den genererade streckkod på skärmen trycker du på Meny och välj "Dela". Precis som med exportera skanningen historia Du kan sedan välja ett program som man kan lagra eller skicka bilden som ett e-postprogram.

Bilden sparas i PNG-format. Observera att en kopia av bildfilen också är skriven till enheten SD-kortet under en som heter katalogen " BarcodeScanner/Barcodes ".
