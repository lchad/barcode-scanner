Barcode Scanner+ kan skanna många andra format, som är mindre vanliga, men fortfarande används i specifika sammanhang som färdbiljetter eller industriella delen märkning. Söker efter alla dessa format kan aktiveras eller inaktiveras i Inställningar, de är funktionshindrade som standard.

PDF417

PDF417 vanligtvis används av företag paketleveransföretag att koda leveransinformation. Observera att det är svårt att läsa PDF417 från en mobil enhet. Du kan välja att aktivera hög upplösning förhandsgranskning i Inställningar, vilket kommer att förbättra avkodning av detta format.

Aztec

Aztec streckkoder finns främst på biljetter för tåg och flygplan.

Industriell 1D format

Slutligen Barcode Scanner+ kan läsa flera olika 1D format som används främst i kommersiella eller industriella applikationer. Ovan är ett exempel på kod 39, de kan vara svåra att skilja visuellt. Ansökan kan läsa:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Najmä môže zobraziť niektoré informácie o čiarových kódov VIN (Vehicle Identification Number).
