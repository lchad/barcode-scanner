Hal ini dimungkinkan untuk memecahkan kode barcode yang disimpan dalam sebuah file gambar pada perangkat Anda, daripada scan dengan kamera perangkat. Untuk melakukannya, tekan Menu dan pilih "Gambar Decode".

Anda akan diminta untuk memilih aplikasi pada perangkat Anda yang menangani gambar, seperti aplikasi Galeri atau file manager. Pilih file gambar yang berisi barcode menggunakan aplikasi tersebut. Anda kemudian akan kembali ke Barcode Scanner+ .

Jika berhasil, Anda akan melihat layar hasil sama seperti jika Anda telah dipindai barcode menggunakan kamera. Jika tidak, Anda akan melihat pesan kesalahan pendek.
