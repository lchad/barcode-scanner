Barcode Scanner+ dapat memindai format lainnya, yang kurang umum tetapi masih digunakan dalam konteks tertentu seperti tiket perjalanan atau pelabelan bagian industri. Scanning untuk masing-masing format dapat diaktifkan atau dinonaktifkan dalam Pengaturan, mereka cacat secara default.

PDF417

PDF417 biasanya digunakan oleh perusahaan pengiriman paket untuk mengkodekan informasi pengiriman. Perhatikan bahwa sulit untuk membaca PDF417 dari perangkat mobile. Anda dapat memilih untuk mengaktifkan tampilan resolusi tinggi dalam Pengaturan, yang akan meningkatkan decoding dari format ini.

Aztec

Barcode Aztec terutama ditemukan pada tiket untuk kereta api dan pesawat terbang.

Industri 1D format

Akhirnya, Barcode Scanner+ dapat membaca format beberapa 1D berbeda yang digunakan terutama dalam aplikasi komersial atau industri. Di atas adalah contoh dari Kode 39, mereka dapat menjadi sulit untuk membedakan secara visual. Aplikasi ini dapat membaca:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Secara khusus, dapat menampilkan beberapa informasi tentang VIN (Vehicle Identification Number) barcode.
