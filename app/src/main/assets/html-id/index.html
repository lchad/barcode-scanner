Barcode Scanner+ menggunakan kamera pada perangkat Anda untuk membaca barcode dari semua jenis. Setelah pemindaian barcode Anda temukan di produk, misalnya, Anda dapat menemukan harga dan ulasan untuk itu. Atau, saat pemindaian baru barcode dua dimensi, seperti QR Codes, Anda dapat mengakses informasi kontak, bookmark, link situs web dan dalam lebih.
