Barcode Scanner+ lahko skenira številnih drugih oblikah, ki so manj pogoste, vendar še vedno uporabljajo v posebnih okoliščinah, kot so vozovnice ali industrijski del označevanja. Skeniranje se lahko za vsako od teh formatov omogočeno ali onemogočeno v Settings, ampak so onemogočena privzeto.

PDF417

PDF417 se običajno uporabljajo podjetja za dostavo paketov za kodiranje ladijskega informacij. Upoštevajte, da je težko brati PDF417 z mobilno napravo. Lahko se odločite, da lahko visoke ločljivosti predogled v nastavitvah, ki bo izboljšala dekodiranje tega formata.

Aztec

Aztec črtne kode so predvsem najti na vozovnice za vlake in letala.

Industrijski 1D formatov

In končno, Barcode Scanner+ lahko preberete več različnih formatov 1D, ki se uporabljajo predvsem v komercialne ali industrijske aplikacije. Zgoraj je primer, Code 39, ki jih lahko težko vizualno razlikovanje. Vlogo lahko preberete:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Še posebej, se lahko prikaže nekaj informacij o VIN (Vehicle Identification Number) črtne kode.
