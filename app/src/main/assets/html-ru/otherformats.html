Сканер штрих-кодов+ может сканировать многие другие форматы, которые являются менее распространенными, но все еще используется в конкретных контекстах, как проездные билеты или промышленной маркировки части. Сканирование для каждой из этих форматов может быть включена или выключена в настройках, они инвалид по умолчанию.

PDF417

PDF417, как правило, используется компаниями комплект поставки для кодирования информации о доставке. Заметим, что это трудно читать PDF417 с мобильного устройства. Вы можете включить предварительный просмотр высокая разрешение в настройках, что позволит улучшить декодирование этого формата.

Aztec

Ацтеков штрих-коды в основном находится на билеты на поезда и самолеты.

Промышленные 1D форматов

Наконец, Сканер штрих-кодов+ можете прочитать несколько различных форматов 1D, которые используются в основном в коммерческих или промышленных целях. Выше пример Code 39, они могут быть трудно отличить визуально. Приложение может читать:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

В частности, он может отображать информацию о VIN (Vehicle Identification Number) штрих-кодов.
