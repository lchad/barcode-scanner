Barcode Scanner+ képes beolvasni sok más formátumok, amelyek kevésbé gyakori, de még mindig használják a speciális környezetben, mint például utazási jegy vagy az ipari részt címkézés. Szkennelés minden ilyen formátumú engedélyezhető vagy letiltható a beállítások, ezek mozgássérült alapértelmezés szerint.

PDF417

PDF417 általában használják csomagszállító cégek szállítási információ kódolására. Ne feledje, hogy nehéz olvasni PDF417 egy mobilkészülék. Választhat, hogy lehetővé teszik a nagy felbontású előnézeti Settings-ben, amely javítja dekódolása ezt a formátumot.

Aztec

Azték vonalkódok elsősorban található jegyeket vonatok és a repülőgépek.

Ipari 1D formátumok

Végezetül, Barcode Scanner+ olvasható több különböző 1D formátumban tárolják, amely elsősorban a kereskedelmi vagy ipari alkalmazásokban. Fent egy példa a Code 39, ők is nehéz megkülönböztetni vizuálisan. Az alkalmazás képes olvasni:

  • Code 39
  • Code 93
  • Code 128
  • ITF
  • Codabar

Különösen képes megjeleníteni néhány információt VIN (Vehicle Identification Number) vonalkódokat.
